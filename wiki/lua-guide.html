Lua Programming for Beginners

この Modpack では、ComputerCraft や周辺モッドを使用して、兵器の自動制御やレーダーシステムの構築が可能です。その心臓部となる Lua 言語を学びましょう。

1. 変数とデータ型

値を保存するための「箱」です。

local ammoCount = 50 -- 数値
local weaponName = "Railgun" -- 文字列
local isActive = true -- 真偽値

2. 条件分岐 (If文)

状況に応じて動きを変えます。

if ammoCount > 0 then
  print("Fire!")
else
  print("Reload necessary!")
end

3. ループ (制御の反復)

索敵や給弾を繰り返す際に使用します。

while true do
  local target = findTarget()
  if target then track(target) end
  sleep(0.1) -- サーバーの負荷を抑えるために重要
end
